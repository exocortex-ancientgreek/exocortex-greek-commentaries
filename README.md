# Exocortex — Ancient Greek commented editions

**52 commented editions of Ancient Greek texts**, turned into a reading interface.

**▶ Read online: https://exocortex-ancientgreek.github.io/exocortex-greek-commentaries/**

## What this is (and is not)

The Greek text and the commentary in these editions are **not mine**. They were written and published — free,
under a Creative Commons licence — by the editors listed below. This repository adds one thing: a way to read
them on a screen.

The printed format ("Text with Facing Vocabulary and Commentary") is excellent on paper and awkward on a
monitor: you cannot tell which words carry a note without scanning the whole apparatus, the vocabulary is a
separate block, and there is no morphology at all. So each page here works differently:

- words that carry a note are **underlined** — you can see there is something to read without looking away;
- **click** one and its note opens beside the word;
- the page's running vocabulary sits in a column next to the text, and the commentary can too;
- **double-click** any word for [Alpheios](https://alpheios.net/) — dictionary and morphology;
- pages turn with ← →, and you can merge several into one spread.

Nothing about the commentary itself is changed. Where our pipeline damaged the anchoring, the page says so —
see *About this edition → Known issues* on any text.

## Known issues

Each edition was OCR'd from the editor's own PDF; a neural pass then anchored every note to the word it
comments on. **The commentary text itself is reproduced as written** — these are faults of our pipeline, not
of the editors. Per-text numbers are in *About this edition* on every page.

| | across these 52 editions |
|---|---|
| Notes anchored to a **line** instead of a word | 2,052 (2.2%) — the neural pass could not match the editor's citation form to a word in the text (`ἀπεφήα(σ)ο` for `ἀπεφήναο`). They still open, from the line number. |
| Multi-word phrases where **not every word** is underlined | 2,215 left, out of 9,509 — the rest were repaired automatically at build time |
| Leftover **beta-code** in a note | 2 editions (Plato's *Symposium*, Herodotus VII), flagged on their own pages |
| **Missing chapter and dialogue headings** | some editions have no section titles yet; being added by hand |
| The editor's **own** slips | occasionally a note cites a line number that is off by one or two. We do not correct these, and **we cannot detect them** — the anchoring is internally consistent either way. |

## How it was made

Each edition was OCR'd from the editor's own PDF, then a neural pass anchored every note to the word it
comments on, then post-processing. That last step is imperfect and the artefacts are measured and shown per
text: notes that ended up on a line instead of a word, multi-word phrases where not every word got underlined,
and leftover beta-code. The numbers are in *About this edition* on each page.

Multi-word phrases are re-anchored automatically at build time, using strict rules — exact match only, a single
unambiguous hit, existing anchors never moved. Roughly three quarters of the broken ones are recovered; the
rest are left alone rather than guessed at.

## Do you make something like this?

**If you publish readers in this format under a free licence, write to me** — I will gladly turn yours into
the same interactive format, and it costs you nothing.

**If you know other such readers, tell me.** These are the ones I know of:

- [C. T. Hadavas](https://www.amazon.com/stores/C.-T.-Hadavas/author/B00MDG04GE) — the same format, a very wide choice of authors (print editions)
- [Patristic Readers](https://patristicreaders.com/gregory-of-nyssa/) — the same format, for the Church Fathers (licensed the same way)
- [Dickinson College Commentaries](https://dcc.dickinson.edu/) — online commentaries (a different approach; better with Alpheios on)

There is also at least one Hayes & Nimis PDF I could not find freely available. If it appears, it will be added.
